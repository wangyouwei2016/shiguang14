# Favicon 设计记录

## 目标

为“拾光 14”生成一个适合浏览器标签栏的小尺寸图标，要求：

- 在 `16x16` 与 `32x32` 下仍有较高识别度
- 保持项目“极简、沉静、侘寂感”的视觉气质
- 与中文品牌名“拾光 14”建立直接关联

## 方案对比

### 方案 A：`拾` 字标（采纳）
- 用单个“拾”字作为主体
- 优点：辨识度高，和品牌名关联最强
- 缺点：更偏中文语境

### 方案 B：`14` 数字标
- 用数字 `14` 作为主体
- 优点：简洁稳定
- 缺点：品牌关联较弱

### 方案 C：抽象光印记
- 用光点、弧线或叶片抽象表达
- 优点：更像品牌符号
- 缺点：小尺寸下不够直观

## 最终方案

采用方案 A：

- 画布：圆角方形
- 背景：`#F5F3EF`
- 主色：`#3A3731`
- 点缀：`#8B7355`
- 图形：居中的“拾”字，使用衬线中文回退字体
- 风格：不使用渐变和复杂阴影，优先保证小尺寸清晰度

## 接入方式

- 新增 `app/icon.svg`
- 在 `app/layout.tsx` 的 metadata 中声明 `icons`
