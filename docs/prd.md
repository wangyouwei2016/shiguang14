产品需求文档 (PRD) - 「拾光 14」极简个人成长管理
1. 产品概述
产品名称： 拾光 14 (暂定名，寓意拾取时光，专注 14 天)

产品定位： 一款为独立创造者、终身学习者打造的极简生活/学习目标管理 Web 应用。

核心理念： 借鉴敏捷开发中“迭代”与“回顾”的精髓，但彻底剥离工作场景中的压迫感术语。通过“灵感无压收集 - 14天短期聚焦 - 每日极简执行 - 成就可视化回顾”的闭环，对抗拖延症，积累“慢慢的成就感”。

2. 用户痛点与解决方案
痛点 1：灵感易逝与目标焦虑。 脑子里想法很多，但缺乏统一记录的地方，或者列了太长的计划看着就焦虑。

解决方案： 设立【灵感池】，仅做收集，不设期限，清空大脑内存。

痛点 2：宏大目标难以执行。 * 解决方案： 设立【14天行囊】，强制只挑出两周内能做的事，小步快跑。

痛点 3：缺乏正反馈，容易半途而废。

解决方案： 设立【时光印记】，将每一次勾选转化为可视化的成长时间轴，提供情绪价值。

3. 核心功能模块说明
3.1 全局组件：闪念捕手 (Quick Capture)
功能描述： 位于应用全局顶部或支持全局快捷键唤出的输入框。

交互规则： 输入文本，按下回车（Enter）即可瞬间将想法存入“灵感池”。支持自动识别并提取 #标签。

3.2 模块一：灵感池 (Idea Pool)
功能描述： 所有未加工想法、长期计划的存放地。

数据结构： 仅包含标题、创建时间、标签（可选）。默认无截止日期。

交互规则： 列表按时间倒序或标签分组排列。支持将条目拖拽至“14天行囊”。

3.3 模块二：14 天行囊 (Focus 14)
功能描述： 用户的两周执行视界。

交互规则： * 提供左右双栏视图，方便从“灵感池”向“行囊”中拖入任务。

每 14 天为一个周期，周期结束时，未完成的任务可选择“放回灵感池”或“顺延至下个 14 天”，系统不作任何红色警告或惩罚提示。

3.4 模块三：今日案头 (Today's Desk)
功能描述： 每日高频使用的极简待办面板。

交互规则：

用户每日从“14天行囊”中挑选 1-3 件事放入案头。

点击任务前的 [ ] 选框，任务文字增加中划线（删除线）并变灰，停顿 1.5 秒后平滑移出当前列表，自动归档至“时光印记”。

3.5 模块四：时光印记 (Time Footprints)
功能描述： 以时间轴形式展示已完成的任务，是提供情绪价值的核心模块。

交互规则：

按日、月倒序展示所有已打勾的任务。

高光时刻 (Highlight)： 允许用户选中某个完成的任务，标记为“⭐高光”，高光任务在时间轴上会放大显示，并支持补充复盘感悟（文本或图片）。

4. 非功能性需求 (NFR)
响应式设计： 优先适配 Web 端和移动端浏览器，方便后续使用跨平台框架（如 Flutter 或 React Native）打包成 Android/iOS 独立 App。

交互动效： 拖拽、打勾、消除等动作必须配备平滑的 Easing 动画，营造“解压感”。
